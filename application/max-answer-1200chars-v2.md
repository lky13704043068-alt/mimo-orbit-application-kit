我正在构建一个面向个人开发者和小团队的 AI Coding Agent 工作流，目标是把需求分析、仓库理解、任务拆解、代码修改、Review、测试修复和文档生成串成一个可持续运行的工程闭环。这个项目重点验证 MiMo 在真实工程任务中的长上下文理解、多文件协同修改和多轮 Agent 工作流稳定性。

目前已完成项目骨架、Agent 工作流设计、Prompt 模板、示例任务和本地评估脚本，并整理成可公开的 GitHub 仓库。项目不是单轮体验，而是持续跑真实 coding-agent 任务：每个任务都会同时读取项目结构、源代码、需求说明、测试日志、错误栈和修复历史，因此对上下文长度和稳定性要求很高。
