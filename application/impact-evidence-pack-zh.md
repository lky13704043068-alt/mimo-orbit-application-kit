# 使用证明与影响力证明补充包

如果申请表允许补充说明，可以填写：

我已经将 MiMo 的申请材料整理为一个公开 GitHub 仓库，包含项目说明、Agent 工作流、Prompt 模板、示例任务、本地评估脚本和证明截图。这个项目会持续用于真实 coding-agent 任务的评估，而不是一次性体验。

当前能提供的证明材料包括：

- GitHub 项目链接
- 项目目录截图
- README 截图
- Git 提交记录截图
- 本地评估报告截图

后续如果获得更高额度，我会继续补充：

- 终端运行日志
- Agent 工作流录屏
- 多模型对比记录
- 真实任务执行样本
- 公开实践文章或 Demo 页面

我的目标是把 MiMo 作为长期的 coding-agent 评估模型来使用，并把有效经验整理成可复用材料，帮助更多开发者理解和接入 MiMo。
